Controle de iluminação residencial remoto via Web

Projeto desenvolvido durante o curso de Tecnlogia em Automação Industrial, que entregou como objetivo o controle e surpevisão de ilumincação de uma residencia de forma remota.

A aplicação web desse projeto permite que o usuário possa monitorar o atual estado da ilumminção dos cômodos de sua residencia, podendo tomar como ação ligar ou desligar a lampada via web
